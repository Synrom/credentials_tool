============
Installation
============


From sources
------------

The sources for credentials_tool can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/Synrom/credentials_tool

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/Synrom/credentials_tool/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/Synrom/credentials_tool
.. _tarball: https://github.com/Synrom/credentials_tool/tarball/master
